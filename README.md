# Legal-Document-Analysis-System

Codebase Submission Note
Due to the large size of the complete codebase and to avoid version control conflict errors on GitHub, we have provided the entire application source code as a compressed ZIP file.

You can access and download the complete project from this Google Drive link: https://drive.google.com/file/d/1otlb4a-4_O2ZIQ-r-H5pFO2S26GYO2Ce/view?usp=sharing
To run the application locally, simply extract the ZIP file, run npm install to install dependencies, add your environment variables to a .env file, and start the development server with npm run dev.
